
## 默认情况下 Xrdp 使用/etc/ssl/private/ssl-cert-snakeoil.key,它仅仅对“ssl-cert” 用户组成语可读。
# 运行下面的命令将 xrdp 用户添加到这个用户组：（缺省未采用ssl）
# sudo adduser xrdp ssl-cert 
 
##重启 Xrdp 服务使得修改生效
```bash
sudo systemctl restart xrdp
```
