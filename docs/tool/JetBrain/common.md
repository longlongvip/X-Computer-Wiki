## IDE 使用的目录

- 配置目录
```bash
# idea.config.path
%APPDATA%\JetBrains\<product><version>
```

- 系统目录
```bash
# idea.system.path
%LOCALAPPDATA%\JetBrains\<product><version>
```

- 插件目录
```bash
# idea.plugins.path
%APPDATA%\JetBrains\<product><version>\plugins
```

## 日志目录
```bash
# idea.log.path
%LOCALAPPDATA%\JetBrains\<product><version>\log
```
